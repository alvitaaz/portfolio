AI CCTV System for Safety Gear and Person Detection

This project involves creating an AI-powered CCTV system using YOLO (You Only Look Once) for real-time object detection. The system is designed to detect people and safety gear, such as safety vests and helmets. By utilizing a Flask API, the CCTV video is streamed via RTSP, and each frame is processed using the YOLO model. The detection results are displayed in real-time through an endpoint accessible via a browser.

Additionally, images of detected individuals are automatically saved to a gallery for future reference. Technologies like OpenCV are used for image processing, while Flask enables the development of a web service to manage video streams and detection results.
